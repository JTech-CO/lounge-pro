개요

이 문서는 Chrome 확장 프로그램 lounge-pro의 개인정보 처리방침입니다.

lounge-pro는 네이버 라운지(lounge.naver.com) 페이지에서 동작하며, 사용자가 직접 지정한 닉네임을 브라우저 로컬 저장소에만 저장합니다. 어떠한 데이터도 외부 서버로 전송하지 않습니다.

1. 처리하는 데이터

  • 차단 닉네임 목록: 사용자가 직접 입력한 닉네임(최대 10개)을 브라우저의 chrome.storage.local에 저장합니다. 이 데이터는 사용자의 기기에만 존재하며 외부로 전송되지 않습니다.
  • 페이지 DOM 읽기: 차단 기능 수행을 위해 현재 열린 네이버 라운지 페이지의 게시글 닉네임 요소를 일시적으로 읽습니다. 또한 프로필 통계 표시를 위해 프로필 페이지의 게시글 수 · 댓글 수 정보를 읽습니다. 읽은 내용은 저장되지 않습니다.
  • 네트워크 요청 감청: 게시글과 작성자의 매핑 정보를 얻기 위해 브라우저 내부에서 네이버 라운지 API 응답을 가로챕니다. 이 과정에서 수집된 데이터는 메모리에만 임시 보관되며, 페이지를 벗어나면 즉시 소멸합니다.
  • 프로필 통계 파싱: 프로필 페이지 방문 시 페이지 내 스크립트 데이터(RSC)에서 총 게시글 수와 댓글 수를 읽어 배지로 표시합니다. 이 데이터는 화면 표시 용도로만 사용되며 저장되지 않습니다.

2. 데이터 공유 및 제3자 제공

lounge-pro는 수집하거나 처리한 데이터를 어떠한 제3자에게도 제공하지 않습니다. 분석 도구, 광고 SDK, 원격 서버, 클라우드 전송 기능이 일절 포함되어 있지 않습니다.

3. 데이터 보관 및 삭제

  • 차단 목록: 사용자가 직접 '해제' 버튼을 누르거나, 브라우저의 확장 프로그램 데이터를 초기화하면 삭제됩니다.
  • 임시 데이터: 페이지 이동 또는 탭 종료 시 메모리에서 즉시 소멸합니다.

추가 안내

  • 보안: 모든 데이터 처리는 사용자의 브라우저 환경 내에서만 이루어집니다.
  • 아동: 특정 연령대를 대상으로 하지 않는 일반 유틸리티 도구입니다.
  • 정책 변경: 기능 변경으로 처리방침이 갱신될 경우 본 페이지의 날짜와 함께 새 내용을 게시합니다.

Overview

This Privacy Policy describes how the Chrome extension lounge-pro handles user data.

lounge-pro operates exclusively on Naver Lounge (lounge.naver.com) and stores only the nicknames you specify — locally in your browser. No data is ever sent to any external server.

1. Data We Process

  • Block list (nicknames): Up to 10 nicknames you enter are saved in chrome.storage.local on your device only. This data never leaves your browser.
  • Page DOM reading: To apply blocks, the extension temporarily reads post-author nickname elements on the active Naver Lounge page. It also reads post and comment count information on profile pages to display stats badges. This data is not stored.
  • Network response interception: To map posts to their authors, the extension intercepts Naver Lounge API responses inside the browser. Any data gathered this way is held only in memory and is discarded immediately upon page navigation or tab close.
  • Profile stats parsing: When visiting a profile page, the extension reads total post and comment counts from in-page script data (RSC) to display as badges. This data is used solely for on-screen display and is not stored.

2. Data Sharing & Third Parties

lounge-pro does not share any data with third parties. The extension contains no analytics, ad SDKs, remote servers, or cloud transfer functionality of any kind.

3. Data Retention & Deletion

  • Block list: Deleted when you press the "해제 (unblock)" button or clear the extension's storage via your browser settings.
  • Temporary data: Cleared from memory instantly upon page navigation or tab close.

Additional Information

  • Security: All data processing is strictly confined to your local browser environment.
