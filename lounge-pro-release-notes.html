v3.0.2 — Forest 테마 및 글쓰기 보호

New Features

  • Forest 테마 신설 (Matrix 대체): 다크 글래스모피즘 기반 숲속 테마
    • 배경 #1a2318(짙은 숲), 카드 rgba(42,56,40,0.75) 반투명 + backdrop-filter: blur(12px)
    • 텍스트 #e8ede6(크림), 포인트 #8da491(세이지 그린), Quicksand 폰트
    • 전체 UI에 둥근 모서리(0.8rem ~ 9999px 캡슐형) 적용, 커스텀 스크롤바
  • 글쓰기 테마 자동 해제 토스트: 화면 전체 변경형 테마(고대비·Forest·Claude Code) 사용 중 글쓰기 진입 시 테마 해제 + 화면 중앙 안내 메시지 2초 표시, 글 등록 후 자동 복원

Improvements

  • 우측 하단 UI 테마 격리 강화: mountUI <style> 블록 전체에 !important 적용, 어떤 테마에서도 차단/테마 패널이 변형되지 않도록 보호
  • 월간 패널 Forest 연동: 다크 글래스 배경 + 캡슐형 프로그레스 바 + 세이지 그린 그라데이션 + Quicksand 폰트

Improvements

  • 다크 브랜드 테마 고정: 라이트/다크 모드 무관하게 #111111 배경 + 네이버 그린(#03C75A) 기반 고정 다크 스타일
  • 그라데이션 테마 테두리 연동: Pastel · Cyberpunk · Holographic · Neon 테마에서 테두리 Scope 활성 시 스탯 패널 테두리에 해당 그라데이션 border-image 적용 (글자·배경은 미변경)
  • 전체 변경형 테마 연동: 고대비(#5) · 매트릭스(#6) 적용 시 패널 와이어프레임화; Claude Code(#7) 적용 시 dashed 테두리 + 모노스페이스 + 터미널 톤으로 전환

v3.0 — 월간 이벤트 트래커

New Features

  • 월간 이벤트 달성 트래커: 프로필 페이지(/profiles/)에서 이번 달 작성한 게시글·댓글 수를 자동 집계하여 프로그레스 바로 표시
    • 달성 목표: 게시글 월 30개, 댓글 월 90개
    • 25% / 50% / 75% / 100% 마일스톤 단계 표시 및 단계별 색상 변화 (게이미피케이션)
    • 100% 초과 시 실제 퍼센티지 + 달성 시머 이펙트
    • 새로고침 버튼으로 즉시 갱신, 5분 캐시로 효율적 동작

Changes

  • injector.js 확장: 활동 API 커서 기반 월별 게시글/댓글 카운팅 엔진 추가 (countMonthlyActivity)
    • API 커서 Base64 디코딩으로 날짜 경계 판별
    • JSON/XML 양쪽 응답 포맷 호환 파서 내장
    • LP_MONTHLY_STATS_REQUEST / LP_MONTHLY_STATS_RESULT 메시지 프로토콜
  • content.js 확장: 프로필 페이지 감지 → 패널 마운트 → 데이터 수신 → 렌더링 파이프라인 추가
  • style.css 재작성: 네이버 라운지 네이티브 디자인 언어 기반 이벤트 트래커 패널 스타일; 테마 엔진과 독립 동작하도록 #lp-monthly-panel 전용 스타일 격리
  • manifest.json 버전 2.6 → 3.0
